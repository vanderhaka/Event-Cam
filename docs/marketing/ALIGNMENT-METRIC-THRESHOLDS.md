# Alignment Metric Thresholds

> Canonical threshold reference for interpreting KPI movement and triggering decisions.

---

## Core Funnel Thresholds

| KPI | Green | Yellow | Red | Action Trigger |
|---|---|---|---|---|
| Visitor → Signup Conversion | On/above target | 5–10% below target | >10% below target | Investigate messaging/funnel friction |
| Signup → Event Creation Conversion | On/above target | 5–10% below target | >10% below target | Audit activation flow and onboarding |
| Event Creation → Paid Conversion | On/above target | 5–10% below target | >10% below target | Audit pricing clarity + checkout friction |
| Upload Participation Rate | On/above target | 5–10% below target | >10% below target | Review guest experience and event prompts |

---

## Loop & Partner Thresholds

| KPI | Green | Yellow | Red | Action Trigger |
|---|---|---|---|---|
| Guest Email Capture Rate | >= target | 5% below target | >10% below target | Validate capture UX and instrumentation |
| Marketing Opt-In Rate | Stable vs baseline | 5% decline | >10% decline | Review consent language and value framing |
| Guest → Creator Conversion | On trend vs cohort target | 10% below trend | >20% below trend | Review nurture and conversion timing |
| Pro Activation Rate | On/above target | 5–10% below target | >10% below target | Review onboarding + referral UX |
| Referral Conversion Rate | On/above target | 5–10% below target | >10% below target | Recheck offer clarity + attribution quality |

---

## Economics Thresholds

| KPI | Green | Yellow | Red | Action Trigger |
|---|---|---|---|---|
| CAC by channel | At/under target | 10–20% above target | >20% above target | Reduce spend or optimize conversion first |
| Payback period | At/under target window | +10–20% slower | >20% slower | Pause scaling and re-evaluate economics |
| Contribution margin per event | Stable/improving | Mild decline | Material decline | Review cost assumptions and pricing posture |

## Alignment Governance Thresholds

| KPI | Green | Yellow | Red | Action Trigger |
|---|---|---|---|---|
| Alignment Score (/20) | 17–20 | 13–16 | <13 | Apply signal playbook response |
| Decision Closure Rate | >=90% on-time | 75–89% | <75% | Escalate decision backlog and owners |
| Doc Sync SLA | >=90% in 48h | 75–89% | <75% | Run doc sync correction sprint |
| Active Experiments | <=5 with clear status | 5 with ambiguous status | >5 or no closeouts | Freeze new tests until normalized |

---

## Threshold Governance Rules

1. Thresholds should be reviewed monthly, not modified ad hoc.
2. Any threshold change requires:
   - decision log entry,
   - changelog update,
   - communication in weekly dashboard.
3. Red-state metrics require explicit owner action within one cycle.
