# Alignment Role Checklists

> First-week checklists by function to ensure alignment work is distributed clearly.

---

## Founder / CEO

- [ ] Confirm final decision authority boundaries (pricing, narrative, channel spend).
- [ ] Approve one primary market for current horizon.
- [ ] Approve one dominant growth loop for the quarter.
- [ ] Approve quarterly message architecture.
- [ ] Sign final path worksheet.

Primary artifact touchpoints:
- `QUESTIONS-THAT-NEED-ANSWERS.md`
- `ALIGNMENT-RACI.md`
- `PATH-ALIGNMENT-WORKSHEET.md`

---

## Marketing Lead

- [ ] Draft and finalize quarterly message architecture.
- [ ] Ensure narrative consistency across all active docs and GTM assets.
- [ ] Align top-of-funnel channels with selected path.
- [ ] Define “not saying this quarter” boundaries.
- [ ] Contribute to weekly dashboard with messaging status signals.

Primary artifact touchpoints:
- `MARKETING-DOCS-REVIEW.md`
- `QUESTIONS-THAT-NEED-ANSWERS.md`
- `ALIGNMENT-DASHBOARD-TEMPLATE.md`

---

## Growth Lead

- [ ] Define KPI owners and weekly KPI cadence.
- [ ] Build active experiment portfolio (max 5 concurrent).
- [ ] Add pass/kill thresholds for each experiment.
- [ ] Maintain weekly alignment scorecard.
- [ ] Report weekly channel efficiency and loop performance.

Primary artifact touchpoints:
- `ALIGNMENT-METRIC-DEFINITIONS.md`
- `ALIGNMENT-SCORECARD.md`
- `ALIGNMENT-MEETING-CADENCE.md`

---

## Product Lead

- [ ] Confirm product priorities align with chosen path (and boundaries).
- [ ] Validate instrumentation needed for KPI definitions.
- [ ] Flag delivery risks for path-critical features.
- [ ] Support experiment implementation requirements.
- [ ] Participate in weekly decision and risk reviews.

Primary artifact touchpoints:
- `PATH-ALIGNMENT-WORKSHEET.md`
- `ALIGNMENT-DASHBOARD-TEMPLATE.md`
- `ALIGNMENT-RUNBOOK.md`

---

## Ops / PMO

- [ ] Maintain artifact hygiene and synchronization.
- [ ] Own decision log integrity and superseded-decision trail.
- [ ] Track documentation sync SLA after decisions close.
- [ ] Run cadence operations (meeting prep, notes, follow-through).
- [ ] Trigger escalations when blockers exceed allowed windows.

Primary artifact touchpoints:
- `ALIGNMENT-DECISIONS-LOG.md`
- `ALIGNMENT-DECISION-CALENDAR.md`
- `ALIGNMENT-ARTIFACTS-INDEX.md`

---

## Finance Owner

- [ ] Approve canonical pricing table and economics assumptions.
- [ ] Validate contribution margin and payback assumptions.
- [ ] Ensure channel spend decisions map to economics guardrails.
- [ ] Review referral economics sustainability.
- [ ] Review monthly economics section in dashboard/reporting.

Primary artifact touchpoints:
- `ALIGNMENT-METRIC-DEFINITIONS.md`
- `ALIGNMENT-PRIORITY0-DECISION-BRIEF.md`
- `ALIGNMENT-DASHBOARD-TEMPLATE.md`

---

## Legal / Compliance

- [ ] Approve consent and email policy baseline.
- [ ] Validate transactional vs marketing communication boundaries.
- [ ] Confirm data retention/deletion policy posture.
- [ ] Review high-risk experiments for compliance exposure.
- [ ] Sign off on compliance-sensitive Priority 0 decisions.

Primary artifact touchpoints:
- `QUESTIONS-THAT-NEED-ANSWERS.md`
- `ALIGNMENT-PRIORITY0-DECISION-BRIEF.md`
- `ALIGNMENT-FAQ.md`

---

## Rule

If any role’s checklist has unresolved critical items after Week 1, path execution should be marked **at risk** until resolved.
