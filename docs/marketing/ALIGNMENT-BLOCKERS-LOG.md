## Escalation SLA

- **High severity:** escalate within 24 hours if unresolved.
- **Medium severity:** escalate within 72 hours if unresolved.
- **Low severity:** review in weekly decision review.

If a blocker affects Priority 0 decisions, treat it as high severity by default.
