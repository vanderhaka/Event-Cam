## Maintenance Rule

Whenever a decision materially changes process, ownership, metrics, or cadence:

1. Add a changelog entry.
2. Update affected artifacts within 48 hours.
3. Link the associated decision ID from `ALIGNMENT-DECISIONS-LOG.md`.
