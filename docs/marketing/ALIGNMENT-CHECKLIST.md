# Alignment Checklist

> Quick execution checklist to confirm strategic alignment is complete and operational.

---

## Phase 1 — Foundation Locked

- [ ] Primary market for next 6 months is decided and documented.
- [ ] Canonical live pricing table is finalized.
- [ ] Pro economics are finalized (created path + referred path).
- [ ] One quarterly message architecture is approved.
- [ ] Compliance baseline for guest capture/email is approved.

---

## Phase 2 — Operating Model Live

- [ ] One dominant growth loop is selected for this quarter.
- [ ] Weekly KPI cadence is scheduled with named owners.
- [ ] Launch playbook has explicit task ownership and due dates.
- [ ] Referral policy includes anti-abuse rules and payout terms.
- [ ] Assumption register exists with confidence labels and validation dates.

---

## Phase 3 — Documentation Synced

- [ ] Contradictory pricing/economic numbers are reconciled across docs.
- [ ] Each strategy doc has owner + last updated date + status label.
- [ ] Decision log is updated with latest closed decisions.
- [ ] Superseded decisions are archived with rationale.
- [ ] Path alignment worksheet is signed by leadership roles.

---

## Phase 4 — Execution Ready

- [ ] Active experiment portfolio is capped and each experiment has pass/kill thresholds.
- [ ] Top 5 risks have owners and mitigation actions.
- [ ] “Not doing this quarter” boundaries are documented.
- [ ] 30/60/90 checkpoints are scheduled.
- [ ] First monthly alignment readout format is agreed and ready.

---

## Final Gate

Only mark “Aligned” when:

- all Priority 0 questions are closed,
- one path is selected and signed,
- canonical pricing is conflict-free across docs,
- weekly KPI and decision cadence is live.
