# Alignment Weekly Operations Checklist

> Tactical weekly checklist to keep alignment disciplined during execution.

---

## Monday — Direction Check

- [ ] Verify no unresolved Priority 0 blockers.
- [ ] Confirm primary path and quarter boundaries remain unchanged.
- [ ] Review open decisions due this week.
- [ ] Confirm owner coverage for all core KPIs.

---

## Tuesday — KPI & Dashboard Prep

- [ ] Update KPI values using canonical metric definitions.
- [ ] Refresh weekly dashboard template.
- [ ] Flag any unexpected metric definition drift.
- [ ] Pre-tag decisions likely needed in KPI standup.

---

## Wednesday — Decision & Experiment Midweek Check

- [ ] Validate each active experiment has current status + threshold.
- [ ] Ensure active experiment count is ≤ 5.
- [ ] Identify experiments requiring scale/kill call this week.
- [ ] Update blockers log for unresolved dependencies.

---

## Thursday — Governance & Sync Hygiene

- [ ] Confirm this week’s closed decisions are in decision log.
- [ ] Confirm docs impacted by closed decisions are synced.
- [ ] Update changelog for material process/ownership shifts.
- [ ] Check scorecard categories at risk and assign fixes.

---

## Friday — Leadership Readout Prep

- [ ] Finalize weekly alignment score.
- [ ] Summarize top drift signal and corrective action.
- [ ] Prepare leadership decisions needed next week.
- [ ] Record next week’s top 3 priorities and owners.

---

## Weekly Exit Gate

Do not mark week as complete unless all are true:

- [ ] Dashboard published.
- [ ] Scorecard logged.
- [ ] Decision log current.
- [ ] Blockers reviewed with SLA compliance.
- [ ] Next-week ownership priorities assigned.
