# Alignment Artifacts Index

This index maps each alignment artifact to its purpose so the team can execute in sequence without confusion.

---

## Core Sequence (Use in This Order)

1. **`ALIGNMENT-START-HERE.md`**  
   - Entry point and minimum alignment exit criteria.

2. **`MARKETING-DOCS-REVIEW.md`**  
   - Strategic critique, risks, prioritization, and recommendations.

3. **`QUESTIONS-THAT-NEED-ANSWERS.md`**  
   - Priority 0/1/2 decision questions plus decision log framework.

4. **`ALIGNMENT-DECISION-CALENDAR.md`**  
   - 30-day decision schedule to close blockers quickly.

5. **`PATH-ALIGNMENT-WORKSHEET.md`**  
   - Convert decisions into one 90-day execution path.

6. **`ALIGNMENT-CHECKLIST.md`**  
   - Final gate to confirm true alignment readiness.

---

## Supporting Operating Artifacts

- **`ALIGNMENT-DECISIONS-LOG.md`**  
  Canonical decision history and superseded-decision audit trail.

- **`ALIGNMENT-MEETING-CADENCE.md`**  
  Weekly/monthly operating cadence to keep alignment active.

- **`ALIGNMENT-SCORECARD.md`**  
  Weekly quantitative health check to detect alignment drift early.

- **`ALIGNMENT-RACI.md`**  
  Ownership clarity for decision rights, execution responsibilities, and escalation paths.

- **`ALIGNMENT-METRIC-DEFINITIONS.md`**  
  Canonical KPI definitions, formulas, cadence, and ownership.

- **`ALIGNMENT-DASHBOARD-TEMPLATE.md`**  
  One-page weekly operating snapshot for leadership and execution teams.

- **`ALIGNMENT-PRIORITY0-DECISION-BRIEF.md`**  
  Structured decision-quality template for resolving path-defining Priority 0 questions.

- **`ALIGNMENT-FAQ.md`**  
  Rapid answers to common alignment and execution questions.

- **`ALIGNMENT-GLOSSARY.md`**  
  Canonical terminology for consistent strategy and KPI language.

## Success Condition

These artifacts are working when:

- Priority 0 decisions are closed,
- one path is selected and signed,
- contradictions across strategy docs are resolved,
- weekly KPI and decision cadence is running.
