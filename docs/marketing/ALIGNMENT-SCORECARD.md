# Alignment Scorecard (Weekly)

> Use this scorecard to quantify whether alignment is strengthening or drifting.
>
> Scoring rule: 0 (not done), 1 (partially done), 2 (fully done).
>  
> Maximum score per week: 20.

---

## Weekly Scoring Table

| Category | Criteria | Score (0/1/2) |
|---|---|---:|
| Strategic Focus | One primary market is still explicit and unchanged without evidence |  |
| Pricing Consistency | No unresolved pricing/economics contradictions across docs |  |
| Message Consistency | One approved quarterly narrative is used across active work |  |
| Decision Hygiene | New decisions logged with owner/date/docs-updated |  |
| KPI Cadence | Weekly KPI review happened with clear outcomes |  |
| Experiment Discipline | Active experiments have pass/kill thresholds and status updates |  |
| Risk Management | Top-5 risk register updated with owners and mitigation status |  |
| Boundary Enforcement | “Not doing this quarter” boundaries were respected |  |
| Document Sync | Strategy docs updated to reflect latest decisions |  |
| Leadership Visibility | Monthly/weekly reporting delivered in agreed format |  |

---

## Interpretation Bands

| Score | Alignment Health | Interpretation |
|---:|---|---|
| 17–20 | Strong | Path is clear, execution disciplined, low drift risk |
| 13–16 | Stable but at risk | Minor drift signals; tighten owner accountability |
| 9–12 | Weak | Conflicting priorities likely; re-run alignment workshop |
| 0–8 | Critical | Alignment failing; pause expansion and reset path decisions |

---

## Mandatory Actions by Score

- **17–20:** continue execution; no structural changes required.
- **13–16:** fix top 2 scoring gaps within 7 days.
- **9–12:** leadership checkpoint within 72 hours; freeze new initiatives.
- **0–8:** immediate reset: close Priority 0 decisions before any new work.
